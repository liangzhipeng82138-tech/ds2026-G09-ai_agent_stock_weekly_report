# AI Agent 自动生成股市周报

> 课程：数据分析与经济决策（ds2026） | 题目：T-E2
> 小组：第 09 组

## 项目简介

本项目旨在构建一个完整的自动化股市周报生成系统：每周自动拉取 A 股和美股的关键数据，调用 AI API 生成摘要文字，结合模板渲染成排版精美的 PDF 周报和 HTML 网页版。
