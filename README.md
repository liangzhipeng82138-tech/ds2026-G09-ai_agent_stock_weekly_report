## 章节分工

本仓库为 **T-E2：AI Agent 自动生成股市周报** 项目的一部分。

### 成员3：黄悦 - 资金流向分析师

**核心定位**：资金是股市涨跌核心驱动力，独立维度、专业性极强。

**具体工作内容**：
- 拆解三大核心资金：北向资金、主力资金、散户资金本周流向数据
- 统计各行业、个股主力净流入/流出榜单，分析资金偏好
- 分析两市成交额变化、量能缩放，判断市场增量/存量博弈状态
- 解读龙虎榜重点个股资金动向、机构买卖情况

**周报固定输出模块**：
- 全市场资金复盘
- 重点资金动向
- 资金偏好分析
